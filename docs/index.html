GitHub Actions を開く Pages 設定を開く

表示できないときの最短手順（2分）

  1. GitHub の Settings → Pages を開く。
  2. Build and deployment を GitHub Actions にする。
  3. Actions タブで最新の Deploy to GitHub Pages が成功しているか確認。
  4. URL https://jonsonpanson114.github.io/book/ を再読み込み。
